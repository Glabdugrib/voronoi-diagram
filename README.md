# Voronoi Diagram

An experimental project focused on implementing and visualizing a custom 2D Voronoi diagram algorithm. The goal is to explore computational geometry concepts by building the algorithm from scratch and rendering the resulting cells through an interactive canvas-based visualizer.

## Requirements

- Node.js 24+
- pnpm

## Getting Started

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

The application will be available through the local Vite development server.

## Technology Stack

- TypeScript
- Vite
- Vue 3
- HTML5 Canvas
- pnpm

No external geometry or Voronoi libraries are used. The algorithm is implemented entirely from scratch for educational and experimental purposes.
